## 📚 레포지토리 컨벤션
 - 자신의 레포지토리를 만들어서 과제를 올리면 됩니다.<br>
 - 레포지토리 이름은 "과제명_이름"으로 지어주세요. ex) TodoList_Jay <br>
 - 스터디 전날까지 올리고, 가능하면 다른 스터디원 코드리뷰도 하면 좋을 것 같습니다.
 - package manage는 npm을 사용하도록 합니다.
